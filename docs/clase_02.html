2. Creando Rangos con pd.cut()

Vamos a dividir a las personas en tres categor√≠as seg√∫n su edad: * Joven: 0 a 30 a√±os. * Adulto: 30 a 60 a√±os. * Adulto Mayor: 60 a 100 a√±os.

  • üêç C√≥digo de Transformaci√≥n
  • üìä Visualizaci√≥n
  • üìù Interpretaci√≥n

Definimos los ‚Äúbordes‚Äù (bins) y las ‚Äúetiquetas‚Äù (labels).

C√≥digo 
# 1. Definimos los cortes: 0-30, 30-60, 60-100
cortes = [0, 30, 60, 100]

# 2. Definimos los nombres de los grupos
nombres = ['Joven', 'Adulto', 'Adulto Mayor']

# 3. Aplicamos la magia de pd.cut
df['grupo_etario'] = pd.cut(df['edad'], bins=cortes, labels=nombres)

# Mostramos el resultado
df[['edad', 'grupo_etario']].head(10)

Ahora que tenemos una variable categ√≥rica (grupo_etario), ¬°podemos usar lo que aprendimos en la Clase 1!

# Contamos cu√°ntos hay en cada grupo
conteo = df['grupo_etario'].value_counts().sort_index()

## 1. El Escenario: Edades de los Encuestados

Seguimos con nuestro proyecto "Muleto". Esta vez nos enfocaremos exclusivamente en la columna **Edad**. A veces el n√∫mero exacto no importa tanto como el "Grupo Etario" al que pertenece la persona.

```{python}
#| label: cargar-datos-c2
import pandas as pd
import matplotlib.pyplot as plt

# Simulamos edades variadas
datos = {
    'encuestado_id': range(1, 11),
    'edad': [18, 22, 55, 60, 15, 82, 33, 29, 45, 19]
}

df = pd.DataFrame(datos)
print("Datos originales:")
print(df['edad'].values)
```

---

## 2. Creando Rangos con `pd.cut()`

Vamos a dividir a las personas en tres categor√≠as seg√∫n su edad:
* **Joven:** 0 a 30 a√±os.
* **Adulto:** 30 a 60 a√±os.
* **Adulto Mayor:** 60 a 100 a√±os.

::: {.panel-tabset}

## üêç C√≥digo de Transformaci√≥n

Definimos los "bordes" (bins) y las "etiquetas" (labels).

```{python}
#| label: aplicar-cut

# 1. Definimos los cortes: 0-30, 30-60, 60-100
cortes = [0, 30, 60, 100]

# 2. Definimos los nombres de los grupos
nombres = ['Joven', 'Adulto', 'Adulto Mayor']

# 3. Aplicamos la magia de pd.cut
df['grupo_etario'] = pd.cut(df['edad'], bins=cortes, labels=nombres)

# Mostramos el resultado
df[['edad', 'grupo_etario']].head(10)
```

## üìä Visualizaci√≥n

Ahora que tenemos una variable categ√≥rica (`grupo_etario`), ¬°podemos usar lo que aprendimos en la Clase 1!

```{python}
#| label: grafico-grupos

# Contamos cu√°ntos hay en cada grupo
conteo = df['grupo_etario'].value_counts().sort_index()

# Graficamos
conteo.plot(kind='bar', color='#003366', rot=0)
plt.title("Distribuci√≥n por Grupo Etario")
plt.ylabel("Cantidad")
plt.grid(axis='y', linestyle='--', alpha=0.5)
plt.show()
```

## üìù Interpretaci√≥n

**An√°lisis:**
Al discretizar la variable, simplificamos el an√°lisis. Observamos que nuestra muestra es predominantemente **Joven** (menores de 30), con una presencia menor de Adultos Mayores.
